📋 ACTIONS À EFFECTUER POUR LE SEO
════════════════════════════════════════════════════════════════════════════════

1. GOOGLE SEARCH CONSOLE - SOUMETTRE LE SITEMAP
────────────────────────────────────────────────────────────────────────────────
   • Allez sur Google Search Console > section Sitemaps
   • Entrez l'URL : https://findmyjobai.com/sitemap.xml
   • Cliquez sur "Envoyer"
   
   Note: Le sitemap est automatiquement généré lors du build npm run build
   et contient toutes les pages statiques + les offres d'emploi dynamiques.

2. GOOGLE SEARCH CONSOLL - VÉRIFIER ROBOTS.TXT
────────────────────────────────────────────────────────────────────────────────
   • Dans Google Search Console > Paramètres > Robots.txt
   • Vérifiez que le fichier est bien détecté :
     https://findmyjobai.com/robots.txt
   
   Le fichier robots.txt autorise l'indexation complète du site et référence
   le sitemap.

3. MICROSOFT CLARITY - TABLEAU DE BORD
────────────────────────────────────────────────────────────────────────────────
   • Accédez à votre tableau de bord Clarity :
     https://clarity.microsoft.com/
   
   • Project ID: xq6fqslzqx
   
   • Vérifiez que le tracking fonctionne (données en temps réel)
   • Consultez les heatmaps et enregistrements de sessions

4. GOOGLE ADSENSE - CONFIGURATION
────────────────────────────────────────────────────────────────────────────────
   • Publisher ID: ca-pub-5351020915477002
   • Les publicités s'affichent pendant le chargement IA (CV, jobs, freelance)
   • Remplacez l'ad slot ID "1234567890" dans AdComponent.jsx par votre vraie unité
   
   Note: Les publicités nécessitent l'approbation AdSense (1-7 jours)

5. META TAGs & OPEN GRAPH - VÉRIFICATION
────────────────────────────────────────────────────────────────────────────────
   • Testez vos meta tags avec :
     - https://developers.facebook.com/tools/debug/
     - https://cards-dev.twitter.com/validator
     - https://www.linkedin.com/post-inspector/
   
   • URL de test: https://findmyjobai.com

6. SCHEMA.ORG - VALIDATION GOOGLE FOR JOBS
────────────────────────────────────────────────────────────────────────────────
   • Testez le markup JobPosting avec :
     https://search.google.com/test/rich-results
   
   • Les offres d'emploi incluent automatiquement le schema.org JobPosting
   • Cela permet l'affichage dans Google for Jobs

7. NETLIFY - DÉPLOIEMENT
────────────────────────────────────────────────────────────────────────────────
   • Le site est déployé sur Netlify
   • URL: https://find-my-job-ai.netlify.app
   • Domaine personnalisé: https://findmyjobai.com
   
   Note: Les déploiements automatiques nécessitent des crédits Netlify.
   En attendant, utilisez les commandes locales :
   
   cd frontend
   npm run build
   # Puis déployez le dossier dist/ manuellement sur Netlify
